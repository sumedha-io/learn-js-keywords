# Learn JavaScript Keywords: var, let, and const

Welcome to this interactive course! An automated bot will check your work via commit comments.

## 📝 Step 1: Initialize your variables
To begin the course:
1. Create a brand new file named `variables.js` in the root folder of this project.
2. Inside that file, use the correct keyword to declare a variable named `birthYear` and assign it your year of birth.
3. Commit and **push** your changes to GitHub!

*Check your commit history on GitHub after pushing to read the bot's review!*
